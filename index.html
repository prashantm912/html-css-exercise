Getting started

Hibernate maps your Java classes to database tables so you can work with objects instead of SQL. Follow the steps below to add it to a project.

  1. Add the Hibernate dependency to your build file.
  2. Create a configuration with your datasource.
  3. Annotate an entity class with @Entity.
  4. Open a session and persist your object.

Why teams pick Hibernate

  • Automatic dirty checking
  • First and second level caching
  • Lazy loading of associations
  • Database portability
